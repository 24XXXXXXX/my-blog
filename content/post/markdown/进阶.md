# Markdown进阶教程

在掌握了Markdown基础语法后，本教程将带你学习更多实用的扩展语法和高级技巧，让你的文档更加丰富多彩。

<!--more-->

## 一、GitHub Flavored Markdown (GFM)

GitHub对标准Markdown进行了扩展，增加了许多实用功能，这些扩展被称为GFM。

### 1. 任务列表

创建待办事项列表：

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个待办事项
  - [x] 子任务1
  - [ ] 子任务2
```

**效果：**

- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个待办事项
  - [x] 子任务1
  - [ ] 子任务2

**应用场景：**
- 项目管理
- 学习计划
- 购物清单
- 工作日程

### 2. 表格高级用法

#### 对齐控制

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :--- | :---: | ---: |
| 内容1 | 内容2 | 内容3 |
| 长内容测试 | 中 | 123 |
```

**效果：**

| 左对齐 | 居中对齐 | 右对齐 |
| :--- | :---: | ---: |
| 内容1 | 内容2 | 内容3 |
| 长内容测试 | 中 | 123 |

#### 表格中的格式

```markdown
| 功能 | 语法 | 示例 |
| --- | --- | --- |
| **粗体** | `**文字**` | **重要** |
| *斜体* | `*文字*` | *强调* |
| `代码` | `` `代码` `` | `console.log()` |
| [链接](/) | `[文字](URL)` | [首页](/) |
```

### 3. 删除线

```markdown
~~这段文字已被删除~~
正常文字 ~~删除的部分~~ 继续正常
```

**效果：**

~~这段文字已被删除~~  
正常文字 ~~删除的部分~~ 继续正常

### 4. 自动链接

GFM会自动将URL转换为链接：

```markdown
访问 https://github.com 了解更多
发邮件到 user@example.com
```

**效果：**

访问 https://github.com 了解更多  
发邮件到 user@example.com

### 5. 表情符号 (Emoji)

使用 `:emoji_name:` 语法：

```markdown
:smile: :heart: :thumbsup: :rocket: :fire:
:tada: :sparkles: :star: :book: :pencil:
```

**效果：**

😄 ❤️ 👍 🚀 🔥  
🎉 ✨ ⭐ 📖 ✏️

**常用表情：**

| 类别 | 表情 | 代码 |
| --- | --- | --- |
| 笑脸 | 😄 | `:smile:` |
| 爱心 | ❤️ | `:heart:` |
| 点赞 | 👍 | `:thumbsup:` |
| 火箭 | 🚀 | `:rocket:` |
| 庆祝 | 🎉 | `:tada:` |

[完整表情列表](https://github.com/ikatyang/emoji-cheat-sheet)

## 二、脚注

脚注用于添加注释或引用来源：

```markdown
这是一段包含脚注的文字[^1]。

这是另一个脚注[^note]。

[^1]: 这是第一个脚注的内容。
[^note]: 这是命名脚注的内容，可以包含多行。
    缩进的内容也属于这个脚注。
```

**效果：**

这是一段包含脚注的文字[^1]。

这是另一个脚注[^note]。

[^1]: 这是第一个脚注的内容。
[^note]: 这是命名脚注的内容，可以包含多行。

**注意：** 脚注会自动显示在文章底部。

## 四、高级代码块

### 1. 语法高亮

支持多种编程语言：

````markdown
```python
def hello_world():
    print("Hello, World!")
    return True
```

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```
````

### 2. 行号和高亮

在Hugo中可以高亮特定行：

````markdown
```javascript {linenos=table,hl_lines=[2,4],linenostart=1}
function calculate(a, b) {
    const sum = a + b;  // 这行会被高亮
    const product = a * b;
    return { sum, product };  // 这行也会被高亮
}
```
````

### 3. 差异对比

使用 `diff` 语言显示代码变更：

````markdown
```diff
function hello() {
-   console.log("Hello");
+   console.log("Hello, World!");
}
```
````

**效果：**

```diff
function hello() {
-   console.log("Hello");
+   console.log("Hello, World!");
}
```

## 六、HTML元素

Markdown支持直接使用HTML标签：

### 1. 文字效果

```html
<u>下划线文字</u>
<mark>高亮文字</mark>
<sup>上标</sup> 和 <sub>下标</sub>
<kbd>Ctrl</kbd> + <kbd>C</kbd>
```

**效果：**

<u>下划线文字</u>  
<mark>高亮文字</mark>  
X<sup>2</sup> 和 H<sub>2</sub>O  
<kbd>Ctrl</kbd> + <kbd>C</kbd>

### 2. 详情折叠

```html
<details>
<summary>点击展开详情</summary>

这里是隐藏的内容。

可以包含任何Markdown语法：

- 列表
- **粗体**
- `代码`

</details>
```

**效果：**

<details>
<summary>点击展开详情</summary>

这里是隐藏的内容。

可以包含任何Markdown语法：

- 列表
- **粗体**
- `代码`

</details>

### 3. 居中对齐

```html
<div align="center">
  <h3>居中的标题</h3>
  <p>居中的段落文字</p>
</div>
```

## 八、链接高级技巧

### 1. 参考式链接

适合多次引用同一链接：

```markdown
这是[链接1][ref1]，这是[链接2][ref2]。

再次使用[链接1][ref1]。

[ref1]: https://example.com "链接1的标题"
[ref2]: https://example.org "链接2的标题"
```

### 2. 自动链接简写

```markdown
[链接文字]

[链接文字]: https://example.com
```

### 3. 锚点链接

跳转到页面内的特定位置：

```markdown
[跳转到第三章](#三高级代码块)

## 三、高级代码块
```

### 4. 脚注链接

```markdown
点击查看[^1]

[^1]: 这是脚注内容
```

## 九、转义字符

当需要显示Markdown语法字符本身时：

```markdown
\*不是斜体\*
\#不是标题
\[不是链接\]
\`不是代码\`
\-不是列表
```

**效果：**

\*不是斜体\*  
\#不是标题  
\[不是链接\]  
\`不是代码\`  
\-不是列表

**需要转义的字符：**

```
\ ` * _ { } [ ] ( ) # + - . ! |
```

## 十四、常见问题

### 1. 表格内容过长怎么办？

使用HTML的 `<br>` 标签换行：

```markdown
| 列1 | 列2 |
| --- | --- |
| 短内容 | 很长的内容<br>可以换行显示 |
```

### 2. 如何在列表中添加代码块？

缩进4个空格或1个Tab：

```markdown
1. 第一步

    ```javascript
    console.log("代码块");
    ```

2. 第二步
```

### 3. 如何嵌套引用？

使用多个 `>` 符号：

```markdown
> 一级引用
>> 二级引用
>>> 三级引用
```
